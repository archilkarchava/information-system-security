Написать программу, которая будет шифовать текст шифром Цезаря, Виженера и Плейфера и расшифровывать введенные сообщения.

Требования:

1. Словарь должен состоять из 500 слов.

2. При запуске программы у пользователя необходимо справишавать, какое действие он хочет совершить: Зашифровать или расшифровать.

Если зашифровать, то каким шифром. Если расшифровать шифр выбирать не дозволяется.

3. Зашифрование и расшифрование должно происходить из файла.

4. Ключ в шифре Виженера может быть любой длины, но не превышать слово из 10 букв.

5. Определение правильности расшифрованного текста должно происходить по словарю. Все слова должны быть проверены по словарюю. Это необходимо для того, чтобы избежать ошибок расшифровки неоднозначных слов.

6. Расшифровка каждого текста должна происходить методом частотного анализа.

СОДЕРЖАНИЕ ОТЧЕТА: 

Отчет по практической работе должен содержать:

1. Описание алгоритмов шифрования.

2. Описание алгоритмов криптоанализа для каждого шифра.

3. Код программы с комментариями.

4. Примеры выполнения работы программы для каждого шифра в части шифрования и расшифрования.

Отчет крепится в систему.

Код защищается на практике.
